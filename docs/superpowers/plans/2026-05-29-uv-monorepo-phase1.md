# UV Monorepo Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transform CoMuRoS + my_a2a from two separate repos with runtime sys.path hacks into a single uv workspace monorepo with standard pip imports.

**Architecture:** `pyproject.toml` at CoMuRoS root declares uv workspace with `my_a2a` as member + workspace path source. `my_a2a` is added as a git submodule. Mini-Agent upgrades from vendored copy + optional dep to git-based mandatory dep. All 4 `sys.path.insert` injection sites are deleted; imports resolve via standard `site-packages`.

**Tech Stack:** uv (Python package manager), hatchling (build backend), git submodule, colcon (ROS 2 build)
