关键概念速查

ROS 2 Service
Topic = 广播电台（发布/订阅，持续流数据）
Service = 打电话（请求/响应，一次性）
Action = 外卖配送（目标/反馈/结果，可取消的长时间任务）
ROS 2 Callback
回调 = "当某事件发生时，ROS 自动调用的函数"
四种类型：订阅回调 / 服务回调 / 定时器回调 / 客户端响应回调
没有 rclpy.spin(node)，节点就像不接电话的人
分工：指挥官 vs 执行者
task_manager = 指挥官，用 LLM 做任务规划（谁做什么）
cleaning_bot_llm = 执行者，用 LLM 做代码生成（怎么走到那个位置）
